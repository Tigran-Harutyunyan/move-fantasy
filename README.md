# The movie db API with Vue 3 + Vite

[Live Demo](https://movie-world-vue.vercel.app)

Features:

- Full responsiveness
- Moviedb API
- Vercel as deployment platform
- Lazy loading images and listings
- Popular people, Movies, Tv shows, Movie details, Person, Search pages
